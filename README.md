# FileNest — Dobby Ads Assignment

A Google Drive-like web application where users can register, create nested folders, and upload images.

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express + TypeScript
- **Database:** MongoDB
- **Storage:** Cloudinary

## Features

- User authentication (Signup / Login / Logout)
- Create nested folders (like Google Drive)
- Upload images inside folders
- Folder size calculation (recursive, includes nested folders)
- User-specific data (users can only see their own folders and images)
